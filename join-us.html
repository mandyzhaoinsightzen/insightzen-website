业务拓展经理 (Business Development)

职位描述

我们现在华东及华北地区招聘业务拓展经理 (Business Development)，寻求在健康医疗、保险、制药等大健康领域进行市场拓展的精英人才。

岗位职责

  1. 负责公司新领域、新业务拓展；
  2. 结合外部竞争环境及公司战略优化业务体系，持续提升公司业务核心竞争力；
  3. 与公司各业务及技术相关部门密切沟通，根据业务发展需求进行流程再造；
  4. 深入战略合作伙伴及客户现场，在合作伙伴与公司内部各部门之间搭建业务运作桥梁，有效传递并满足客户需求；
  5. 与合作伙伴保持长期良好沟通，推动合作业务能力落地。

任职要求

  1. 研究生及以上学历，计算机、工商管理或医疗相关专业；
  2. 10年以上行业业务拓展经验；
  3. 熟悉业务各环节，能依据业务需求提供拓展方案；
  4. 具备良好的沟通能力及影响力；
  5. 积极心态，有大局观。

加分项

  1. 对大数据及人工智能领域有了解和工作经验；
  2. 具有健康医疗、保险、制药等垂直领域的行业认知；
  3. 英语读写听说能力强。

数据科学家（Data Scientist）

职位描述

我们正在寻找具有资深机器学习经验的数据科学家，参与构建为行业研究服务的主要机器学习算法和平台。候选者需要定期搜索，阅读，评估顶级研究论文以及开源项目，建立并执行计划，将他们的方法和源代码整合到我们的研究过程和平台中。

任职要求

  1. 博士/硕士学位，计算机科学、统计学、数学、运筹学、经济学、金融学、物理学、心理学及相关专业；
  2. 具备针对以下模型及其优化算法的经验（至少一项）：
  • 图形模型，例如：条件随机场、 贝叶斯网络
  • 遗传算法
  • 一阶和基于命题规则的系统，例如：可跟踪马尔可夫逻辑、Prolog、Lifted Inverse Deduction 算法
  • 循环和递归神经网络，例如：LSTM、GRU
  • 自然语言处理，例如： 自动生成文本、自动文本摘要
  • 强化学习
  • 决策树（集成型）
  • 基于实例的学习，例如：SVM、k-nearest neighbor、协同过滤推荐系统
  • 时间序列分析，例如： Co-integration、VAR等模型

加分项

  1. 熟悉流行机器学习平台和软件包（如tensorflow，azure，scikit-learn等）；
  2. 良好的英语口头和书面沟通能力。

联系方式：HR@insightzen.com

数据工程师（Data Engineer）

职位描述

我们目前正在招募充满激情的数据工程师，您将作为知盛数据全球开发团队的一员，进行开发、优化和运营在针对防止潜在欺诈风险并提供智慧洞察的数据管道和应用上的工作。在这个岗位上，您将有机会参与建立可伸展、高效、自动化的大规模数据流水线处理流程，以及构建基于开放式数据池的应用。

岗位职责

  1. 建立基础设施，收集大量原始数据并处理成适合分析的数据；
  2. 在大量数据的基础上设计和实施近实时数据应用程序；
  3. 提高查询/监控/分析等数据产品的能力，降低普通使用者在大数据环境下的工作量；
  4. 支持业务团队的操作监控请求；
  5. 跟踪风险操作问题并与合作伙伴一起研究和处理。

任职要求

  1. 硕士/学士学位，在相关领域拥有1-2年的工作经验；
  2. 熟练掌握Java，Scala或C ++编程技能;熟练使用Python，Linux Bash或Perl其中至少一种脚本语言；
  3. 具有定制数据管道设计，实施和维护方面的项目经验；
  4. 具有大数据平台/编程经验（Hadoop / MapReduce / Spark / Hive / HBase / Kafka / Flume / Flink），特别是Spark和Hbase；
  5. 具备使用Storm或Spark-Streaming等解决方案构建流处理系统的经验；
  6. 具有数据库使用经验（Oracle / MySQL / MongoDB / Redis），精通SQL数据库和SQL编程。

加分项

  1. 具有网络开发和机器学习应用方面的经验；
  2. 良好的英语口头和书面沟通能力。

联系方式：HR@insightzen.com

岗位职责

  • 全面制定并执行集团及下属子公司市场营销计划，通过线上、线下活动、搜索、媒体、在线营销、事件、社交等有效的方式促进客户需求（client leads）的快速增长；
  • 负责面向客户的产品市场推广，负责市场活动策划、营销方案的制定、营销方案落地等工作的整体安排及执行；
  • 负责市场研究和市场分析，发掘和引领市场需求；
  • 通过有效数据支持分析市场动态、行业竞争环境和制定驱动业务增长的营销策略；
  • 维护和提升外部合作伙伴关系，通过影响既有和潜在的合作伙伴，为公司赢得更多的渠道机会；
  • 负责部门内部资源的整合，以及公司各事业部之间的沟通与协作。

任职要求

  • 本科及以上学历；
  • 8年以上市场营销相关经验；具有丰富B2B营销经验优先；
  • 熟悉企业级产品营销各环节，能够根据业务需求灵活运用整合营销的各种方式；
  • 具备良好的沟通能力及影响力；
  • 具备英语读写听说能力。

加分项：

  • 具有外企工作经验优先；
  • 对大数据及人工智能领域有了解和工作经验；
  • 对保险、健康医疗、制药等领域有了解；
  • 善于从数据中获得洞察力。
